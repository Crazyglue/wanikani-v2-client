# WaniKani Client

## Project overview

This project is an API client for the [WaniKani API](https://docs.api.wanikani.com/20170710/#introduction). It is compatible in both the browser and node.js runtimes (Node 20+).

## Workflow Best Practices

### Debugging Tips

- Use `npm run lint` to lint the codebase
- Use `npm run fmt:check` to check for formatting issues
- Use `npm run fmt` to automatically format the code
- Use `npm run build` to build the codebase
- Use `npm run test` to run unit tests
